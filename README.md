# OptiFlows

Функция NetMatrix.m генерирует случайную матрицу структуры сети, пропускной способности и величины информационных потоков от узла к узлу. 
Цель - моделирование случайных структур сетей для дальнейших математических исследований.
Исходные данные функции: количество узлов и максимальная пропускная способность.
Результат функции: массив случайной матрицы сети с пропускными способностями, массив случайных потоков потоков.
